# DevelopersHub Advanced Data Science & Analytics Internship

## Overview

This repository contains my solutions for the DevelopersHub Corporation Advanced Data Science & Analytics Internship tasks. The projects focus on real-world data science applications including machine learning, customer segmentation, explainable AI, forecasting, and business intelligence.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## Completed Tasks

### Task 1: Term Deposit Subscription Prediction

**Objective:** Predict whether a bank customer will subscribe to a term deposit.

**Key Activities:**

* Data preprocessing and feature encoding
* Logistic Regression and Random Forest classification
* Model evaluation using Confusion Matrix, F1-Score, and ROC Curve
* Explainable AI using SHAP

**Skills Gained:**

* Classification Modeling
* Feature Engineering
* Explainable AI (XAI)
* Customer Behavior Analysis

---

### Task 2: Customer Segmentation Using Unsupervised Learning

**Objective:** Segment customers based on spending behavior and income.

**Key Activities:**

* Exploratory Data Analysis (EDA)
* K-Means Clustering
* PCA-based cluster visualization
* Customer segment interpretation and marketing strategy recommendations

**Skills Gained:**

* Unsupervised Learning
* Customer Segmentation
* Dimensionality Reduction
* Data-Driven Decision Making

---

### Task 3: Energy Consumption Time Series Forecasting

**Objective:** Forecast household energy consumption using historical data.

**Key Activities:**

* Time-series preprocessing
* Feature engineering
* ARIMA, Prophet, and XGBoost model comparison
* Forecast visualization and performance evaluation

**Skills Gained:**

* Time Series Forecasting
* Feature Engineering
* Model Evaluation
* Temporal Data Analysis
